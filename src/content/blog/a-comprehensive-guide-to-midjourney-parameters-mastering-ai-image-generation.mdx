## Understanding Midjourney Parameters

Parameters are options added to a prompt that change how an image generates. They can alter aspect ratios, switch between model versions, change upscalers, and much more. Parameters are always added to the end of a prompt, and you can use multiple parameters in each prompt.

```shell
/imagine prompt: A serene lake at sunset --ar 16:9 --q 1 --v 5
```

In this example, we're using three parameters: aspect ratio, quality, and version.

---

## Basic Parameters

Let's explore some of the most commonly used parameters:

### 1. Aspect Ratio (--ar or --aspect)

Changes the aspect ratio of the generated image.

```shell
/imagine prompt: Cyberpunk cityscape --ar 2:1
```

### 2. Chaos (--chaos)

Influences how varied the results will be. Higher values produce more unusual and unexpected generations.

```shell
/imagine prompt: Abstract emotion --chaos 50
```

### 3. Quality (--q or --quality)

Determines rendering quality and time. Values can be .25, .5, 1, or 2 with 1 being the highest quality. A new 'quality' parameter (`--q`) has been introduced in 6.1. Using `--q 2` can potentially increase texture at the cost of reduced image coherence and longer generation time.

```shell
/imagine prompt: Detailed landscape --q 1
```

### 4. Seed (--seed)

Sets a specific seed for reproducibility. Seeds are integers between `0` and `4294967295`.

```shell
/imagine prompt: Futuristic space station --seed 1234
```

### 5. Stop (--stop)

Finishes a job partway through the process, resulting in less detailed images. Values range from `10` to `100`.

```shell
/imagine prompt: Misty forest --stop 80
```

---

## Advanced Parameters

### 6. Style (--style)

Switches between different model versions or applies specific styles.

```shell
/imagine prompt: Anime character --style cute --niji 5
```

### 7. Stylize (--s or --stylize)

Influences how strongly Midjourney's default aesthetic style is applied. Values range from `0` to `1000`, with `100` as default.

```shell
/imagine prompt: Portrait of a wise old man --s 750
```

### 8. Tile (--tile)

Generates images that can be used as repeating tiles for seamless patterns.

```shell
/imagine prompt: Art deco pattern --tile
```

### 9. Weird (--weird or --w)

Explores unusual aesthetics. Values range from `0` to `3000`.

```shell
/imagine prompt: Surreal dreamscape --w 1000
```

### 10. Version (--v or --version)

Specifies which version of the Midjourney algorithm to use.

```shell
/imagine prompt: Futuristic cityscape --v 5.2
```

### 11. Uplight (--uplight)

Uses a lighter upscaling algorithm, resulting in faster processing but potentially less detail.

```shell
/imagine prompt: Futuristic cityscape --v 5.2

Uses a lighter upscaling algorithm, resulting in faster processing but potentially less detail.
```

### 12. No (--no)

Specifies elements to exclude from the generated image.

```shell
/imagine prompt: Fantasy dragon --no fire
```

### 13. Video (--video)

Generates a video showing the image generation process from start to finish. Note: This may increase processing time.

## Model Version Compatibility

Different Midjourney model versions support various parameters. Here's a quick overview of compatibility for some key parameters:

| Parameter | Version 6.1          | Version 6            | Version 5.2          | Niji 5               |
| --------- | -------------------- | -------------------- | -------------------- | -------------------- |
| Chaos     | 0-100                | 0-100                | 0-100                | 0-100                |
| Quality   | .25, .5, 1, 2        | .25, .5, 1           | .25, .5, 1           | .25, .5, 1           |
| Stylize   | 0-1000 (default 100) | 0-1000 (default 100) | 0-1000 (default 100) | 0-1000 (default 100) |
| Weird     | 0-3000 (default 0)   | 0-3000 (default 0)   | 0-1000 (default 0)   | 0-1000 (default 0)   |
| Uplight   | Yes                  | Yes                  | Yes                  | No                   |
| No        | Yes                  | Yes                  | Yes                  | Yes                  |
| Video     | Yes                  | Yes                  | Yes                  | No                   |
| Realtime  | Yes                  | Yes                  | No                   | No                   |
| Upload    | Yes                  | Yes                  | Yes                  | Yes                  |

## Tips for Effective Parameter Use

- Experiment with different parameter combinations to find what works best for your style.
- Remember that some parameters affect initial generation, while others impact variations and remixes.
- Use the `--seed` parameter to recreate or iterate on specific results.
- Adjust the `--stylize` parameter to find the right balance between Midjourney's style and your prompt.
- Utilize the `--quality` parameter to balance between speed and detail in your generations.
