An interactive guided tour showcasing Alteryx's suite of products and capabilities.

## Project Overview

The Alteryx Guided Tour provides an immersive, interactive experience for users to explore Alteryx's data analytics platform. The tour guides users through key features and capabilities, demonstrating the power of Alteryx's solutions through hands-on examples.

## Technical Details

- React-based interactive interface
- Parcel for build optimization
- Sitecore CMS integration
- Custom animation system
- Analytics integration
- Performance optimization

## Key Features

- Step-by-step product walkthroughs
- Interactive feature demonstrations
- Progress tracking
- Custom animation sequences
- Responsive design
- Analytics tracking
