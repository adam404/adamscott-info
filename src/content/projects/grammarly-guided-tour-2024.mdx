# Grammarly Guided Tour 2024

A comprehensive branching tour of Grammarly's features per vertical, built with Next.js, TypeScript, and Tailwind, incorporating unit testing for robust performance.

## Project Overview

The Grammarly Guided Tour 2024 is an interactive experience that showcases Grammarly's features across different verticals. The application uses a sophisticated branching system to personalize the tour based on user interests and needs.

## Technical Details

- Built with Next.js and TypeScript for type safety and performance
- Implemented custom routing for complex branching paths
- Used Tailwind CSS for responsive design
- Incorporated comprehensive unit testing
- Automated deployment and testing pipelines

## Key Features

- Interactive feature demonstrations
- Personalized user journeys
- Responsive design across devices
- Performance optimized animations
- Comprehensive analytics tracking
