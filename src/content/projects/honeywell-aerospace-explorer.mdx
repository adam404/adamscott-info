A Next.js application with Contentful integration, featuring offline capabilities and a Tailwind-styled interface for exploring Honeywell's aerospace products.

## Project Overview

The Honeywell Aerospace Explorer is a sophisticated web application designed to showcase and explore Honeywell's aerospace product line. The application features offline capabilities, ensuring accessibility even in environments with limited connectivity.

## Technical Details

- Next.js framework with SSR/SSG capabilities
- Contentful headless CMS integration
- Offline-first architecture
- Tailwind CSS for responsive design
- Custom API development

## Key Features

- Comprehensive product catalog
- Offline functionality
- Advanced search capabilities
- Interactive product visualizations
- Real-time content updates via Contentful
